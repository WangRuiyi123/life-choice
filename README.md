# life-choice
🎀 LifeChoice.exe - 生活选择盲盒网页
复古 Win98 风格 · 点开即玩 · 专治选择困难症
✨ 这是什么？
一个像素风复古盲盒网页，打开就能随机抽取生活小建议：
- 不知道吃什么？抽！
- 不知道干嘛放松？抽！
- 想找点乐子/提升效率？抽！

内置 6 大类盲盒，每次打开都是小惊喜：
- 🍜 生活选择盲盒
- 🧸 放松治愈盲盒
- 🤪 搞笑随机盲盒
- ⚡ 效率提升盲盒
- 🎨 创意灵感盲盒
- 🌸 运动健康盲盒

🎮 怎么玩？
1. 打开网页 → 选择分类
2. 点击「抽盲盒」
3. 等待进度条加载 → 开出你的今日小任务
4. 不满意？再来一次！

纯前端网页，无需安装、无需登录，点开即玩。

🖥️ 预览风格
- 经典 Win98 窗口像素风
- 复古按钮、进度条、标题栏
- 绿色终端文字 + 闪烁动画
- 手机/电脑自适应，随时随地开盲盒

📁 文件说明
- `index.html` - 主页面 + 样式 + 所有盲盒内容
- 无外部依赖，直接双击即可本地运行

🌟 适合场景
- 纠结星人日常选择
- 摸鱼放松小工具
- 朋友圈整活互动
- 送给朋友的小礼物网页


> 生活太复杂？交给盲盒帮你选！🎁
